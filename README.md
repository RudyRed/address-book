# react-boilerplate

This is a pure React client-side app (SPA) boilerplate.

## Features

- React 16.x
- Webpack 4.x
- Babel 7.x
- ESLint 6.x
- React Router 5.x
- React Helmet
- Material UI 4.x
- Prettier setup
- Testing setup with Jest and React Testing Library

## Install

```
yarn install
```

## Run for dev

```
yarn start
```

## Build for production

```
yarn build
```
